# WinBeacon — lecture simple

Alternative claire à l'Observateur d'événements Windows. **Application WinUI 3** pensée pour les utilisateurs novices : langage clair, actions visibles, détails techniques repliés.

## Produit

- **Nom** : WinBeacon
- **Éditeur** : Marco Barbaro
- Projet principal : `EventViewer.WinUI` (+ `EventViewer.Core`)
- Legacy WPF : voir `LEGACY_WPF.md`

## Chemins

- État : `%LocalAppData%\WinBeacon\State`
- Exports : `%UserProfile%\Documents\WinBeacon\Exports`
